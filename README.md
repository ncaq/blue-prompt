# blue-prompt

LLMにブルーアーカイブを扱わせる時に役に立つリポジトリ。
それを目指しています。

## 想定利用環境

### LLMサービス

自分が有料部分をそれなりに利用できるLLMサービスは現在(2026年1月24日)は以下の通りです。

- Claude
- GitHub Copilot Chat
- Gemini

その中でもClaudeが私の環境では一番使えるようになっているので、
主にClaudeを対象にすることになります。

しかし自然言語のデータが多めなので、
普通のLLMでもある程度は使えると思います。

### 典型的な使い方

ClaudeのProjectを作成して、
このGitHubリポジトリをファイルとして追加して、
必要なリソースを選択することを想定しています。

必要なリソースはその時次第で選択してください。
